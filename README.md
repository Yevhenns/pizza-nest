## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
npm install
```

## Running the app

Copy .env.example, rename it to .env and fill the environment variables.

# development

```bash
npm run start
```

# watch mode

```bash
npm run start:dev
```

# production mode

```bash
npm run start:prod
```
